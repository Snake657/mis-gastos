Fuentes y metodología: Reservas brutas: API oficial del BCRA (Estadísticas Monetarias v3.0, idVariable 1). Si CORS falla, fallback a serie mensual de datos.gob.ar. Tasa: API oficial del BCRA (Monetarias) — se prueban variables 6/7/34/35/46 hasta encontrar la que responde. Vencimientos: OPC · Secretaría de Finanzas · FMI MFSM. 2024–2026 reportados; 2027–2032 estimados (pueden variar por refinanciaciones, canjes y nuevos acuerdos). Inflación, riesgo país y UVA: ArgentinaDatos (open source, sin API key). Desocupación y pobreza: datos oficiales de INDEC (EPH), curados manualmente en el código (la API de datos.gob.ar para estas series fue inestable). Para actualizarlos: editar las constantes DESOC_DATA y POBREZA_DATA en el HTML. Esta herramienta es informativa y no constituye asesoría financiera.
